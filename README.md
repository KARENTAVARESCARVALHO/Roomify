Bem-vindo ao RoomFy
É muito bom ter você aqui no nosso projeto! Este documento fornece informações específicas sobre o projeto RoomFy, que consiste em um SISTEMA DE RESERVA DE SALAS, para ajudar você a entender, resumidamente, como ele funciona e em quais objetivos ele se baseia.
1. Introdução
A gestão eficiente de salas de aula é um desafio enfrentado por diversas instituições de ensino. A ausência de um sistema centralizado pode gerar conflitos de horários, reservas duplicadas e dificuldades na organização dos espaços disponíveis, além de muitas salas vazias quando poderiam muito bem estarem sendo utilizadas com um propósito.
Com o objetivo de otimizar esse processo, foi proposto o desenvolvimento de um Sistema de Reserva de Salas, permitindo que professores e coordenadores realizem agendamentos de forma simples, rápida e segura.
2. Objetivo Geral
Desenvolver uma aplicação web destinada ao gerenciamento de reservas de salas institucionais, proporcionando maior organização dos ambientes e controle dos agendamentos.
Objetivos Específicos
Permitir o login de usuários autorizados.Exibir salas disponíveis para reserva.Permitir a criação e o cancelamento de reservas.Evitar conflitos de horários.Garantir segurança por meio de controle institucional de acesso.
3. Público-Alvo
O sistema será destinado exclusivamente a:
ProfessoresCoordenadoresAdministração da instituição
Não haverá cadastro público de usuários.
4. Regras de Negócio
Apenas usuários cadastrados pela administração poderão acessar o sistema.Não será permitido cadastro livre de usuários.Professores e coordenadores receberão login e senha fornecidos pela administração.O usuário poderá alterar sua senha após o primeiro acesso.Uma sala não poderá ser reservada para horários já ocupados.O usuário poderá cancelar reservas realizadas por ele.O sistema deverá apresentar apenas horários disponíveis para agendamento.
5. Requisitos Funcionais
RF01 – Realizar login no sistema.RF02 – Alterar senha, se quiser, após o primeiro acesso.RF03 – Visualizar salas disponíveis.RF04 – Consultar horários disponíveis.RF05 – Criar reservas.RF06 – Cancelar reservas.RF07 – Consultar reservas realizadas.RF08 – Exibir painel principal com informações resumidas.
6. Requisitos Não Funcionais
RNF01 – Interface simples e intuitiva.RNF02 – Tempo de resposta inferior a 5 segundos.RNF03 – Sistema acessível por navegador web.RNF04 – Armazenamento seguro das senhas.RNF05 – Compatibilidade com aparelhos modernos.
